XImage (Extensible Image) is an open technology for real-time image manipulation over HTTP. It defines a standard interface for performing common image operations such as resizing, cropping, compressing, filtering, etc. In addition, it describes how to add extended functionality that might be custom for your use case in a manner that is discoverable and reusable.

QUERY STRING

help Shows this help screen. Each implementation of XImage must show this help screen which is primarily used to show which features it supports.
w Width of the image. If h is not supplied, it will be inferred proportionally. By default, it will not be resized larger than its original dimensions. To allow upscaling append '!'. pink.jpg?w=100
h Height of the image. If w is not supplied, it will be inferred proportionally. By default, it will not be resized larger than its original dimensions. To allow upscaling append '!'. pink.jpg?h=100
f Filters, functions and effects to apply to the image. Combine more than one of these together using ','.
o Output format of the image.
