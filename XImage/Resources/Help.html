XImage (Extensible Image) is an open technology for real-time image manipulation over HTTP. It defines a standard interface for performing common image operations such as resizing, cropping, compressing, filtering, etc. In addition, it describes how to add extended functionality that might be custom for your use case in a manner that is discoverable and reusable.

  • help

    Shows this help screen. Each implementation of XImage must show this help screen which is primarily used to show which features it supports.

    • pink.jpg?help

  • w

    Width of the image. If h is not supplied, it will be inferred proportionally. Dimensions will not be resized larger than the original unless upscaling is indicated by appending '!'.

    • pink.jpg?w=100

  • h

    Height of the image. If h is not supplied, it will be inferred proportionally. Dimensions will not be resized larger than the original unless upscaling is indicated by appending '!'.

    • pink.jpg?h=100

  • f

    Filters, functions and effects to apply to the image. Combine more than one of these together using ',' or ';'.

      {{filters}}

  • o

    Output formats. These can be compression codecs, diagnostic details, even this very help page, etc.

      {{outputs}}
